# Calculadora de Series de Fourier Trigonométricas

Aplicación de escritorio para cálculo y visualización de Series de Fourier.

**Autor:** Demian, grupo 5CM1  
**Materia:** Procesamiento de Señales Digitales - ESCOM

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python main.py
```

## Características

- Definición visual de funciones a trozos
- Cálculo simbólico con SymPy
- Control de periodo (dominio, infinito, personalizado)
- Espectro de frecuencias (forma compleja)
- Visualización de armónicos individuales

## Archivos principales

- `main.py` - Interfaz gráfica principal
- `fourier_engine.py` - Motor de cálculo matemático
- `parser_funcion.py` - Parser de expresiones
- `piecewise_widget.py` - Widget de funciones a trozos
- `visualizacion_armonicos.py` - Ventana de análisis detallado
- `ejemplos.py` - Señales predefinidas
